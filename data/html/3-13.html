Amazon was one of the first providers of cloud computing. One year after the beta release of EC2 in 2006,

two new instance types (Large and Extra-Large) were added, followed in 2008 by two more types, High-

CPU Medium and High-CPU Extra-Large. New features include static IP addresses, availability zones,

and user-selectable kernels as well as the Block Store (EBS). Amazon EC2 has been in full production

mode since October 2008 and supports an SLA and the Microsoft Windows operating system as well as

the Microsoft SQL Server.
