A logical clock (LC) is an abstraction necessary to ensure the clock condition in the absence of a global

clock. Each process pi maps events to positive integers. Call LC(e) the local variable associated with

event e. Each process time stamps each message m sent with the value of the logical clock at the time of

sending, T S(m) = LC(send(m)). The rules to update the logical clock are specified by the following
